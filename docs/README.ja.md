## 🎯 重要な概念

### 1. マルチエージェントシステム

TradingAgentsは複数の専門エージェントが協力して動作します：

```
アナリストチーム → リサーチャーチーム → トレーダー → リスク管理 → 最終判断
```

各エージェントは独立して動作し、LLMを使用して分析を行います。

### 2. LangGraph

LangGraphは、エージェント間の情報フローを管理するフレームワークです。

**主要な概念**:
- **ノード**: 各エージェント
- **エッジ**: エージェント間の接続
- **状態**: 共有される情報

### 3. メモリシステム

ChromaDBを使用して、過去の取引判断と結果を保存します。

**利点**:
- 過去の経験から学習
- 類似の状況で参照
- 判断の改善

### 4. データフロー

外部APIからデータを取得し、エージェントに提供します。

**データソース**:
- FinnHub: ニュース、センチメント
- Yahoo Finance: 株価、財務諸表
- Reddit: ソーシャルメディア
- Google News: ニュース記事
